# Security Policy

## Supported Versions

This project is a **research and educational prototype** created as part of a Bachelor's thesis at the University of Zurich.  
It is **not intended for production use**, commercial deployment, or real-world security decision-making.

Only the latest commit on the `main` branch is maintained for research visibility.

## Scope of Security Considerations

This repository is **not a production-ready phishing detector**.  
The following are **explicitly out of scope**:

- Protection of real users, accounts, or infrastructure  
- Deployment in enterprise settings  
- Handling or storage of sensitive personal data  
- Adversarial robustness guarantees  
- Real-time email filtering  
- Guarantees against model inversion or extraction  
- Use as a cybersecurity product
